# Grafana_dashbaord

Stripped down monitoring dashboard for RPI individual use.

# Stack

Containerized grafana, influxdb and telegarf.

# Telegraf Configs

Telegraf configs are used to capture CP temperature with exec plugin, DNS query and ping average ms.
